Version #: v0.0.02
This Repo is dedicated for easy movement of files when needed, along with simple backups.
As of now, there are simple presets for a Minecraft Bedrock server, and a Minecraft Java server compatible with console players.

Docker images to be used:
  https://hub.docker.com/r/itzg/minecraft-server --- Java ; Java/Bedrock
  https://hub.docker.com/r/itzg/minecraft-bedrock-server --- Bedrock

Notes:
  - Add image - https://hub.docker.com/r/itzg/mc-backup ; back-ups the server(s)
  - Add image - https://hub.docker.com/r/ccarney16/pterodactyl-panel ; Dashboard/panel for quicker game server setup and management
  - (IF NEEDED) Add image - https://hub.docker.com/r/filebrowser/filebrowser ; Browse specified files in a web ui
  - Add image - https://hub.docker.com/r/tailscale/tailscale ; To create a vpn to remotely access services anywhere
  - Add image - https://hub.docker.com/r/portainer/portainer ; Web ui to manage containers and stacks, along with monitoring statistics and logs
